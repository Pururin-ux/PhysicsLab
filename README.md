# PhysicsLab

Тренажёр по физике для подготовки к ЦЭ/ЦТ (Беларусь): короткие тренировки
по 10 задач, разбор типичных ошибок, честный прогресс без выдуманных
процентов готовности.

## Структура

- `apps/web` — продукт (Next.js 15 + React 19 + Tailwind v4). Генератор
  задач — в `apps/web/lib/server/task-generator`.
- `legacy/` — архив прошлых прототипов (CRA, Astro, ранние версии).
  Хранится в git выборочно: без артефактов сборки и тяжёлых PDF.
- Корневой `package.json` — только оркестрация: команды делегируются
  в `apps/web`.

## Установка

Нужны две установки — корень содержит только Playwright и TypeScript,
зависимости приложения живут в `apps/web`:

```bash
npm ci
npm ci --prefix apps/web
```

## Команды (из корня)

| Команда | Что делает |
| --- | --- |
| `npm run dev` | dev-сервер на :3000 |
| `npm run check` | tsc --noEmit |
| `npm test` | все юнит-тесты (node --test) |
| `npm run test:physics` | тесты физядра и генератора |
| `npm run test:e2e` | Playwright (сам поднимает сервер) |
| `npm run build` | production-сборка |
| `npm run preview` | next start поверх сборки |

## Известные грабли

- **Не запускайте `npm run build`, пока работает `npm run dev`** — они
  делят `.next/`, и dev-сервер начнёт сыпать `ENOENT _buildManifest`.
  Остановите dev, соберите, потом запускайте снова.
- E2E реиспользуют сервер на :3000, если он уже запущен. Если порт занят
  чем-то посторонним — освободите его перед `npm run test:e2e`.
- Страницы `/dev/*` (витрины графиков и диаграмм) доступны только в dev;
  в production-сборке весь сегмент отвечает 404.
